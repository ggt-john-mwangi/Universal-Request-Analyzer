Getting Started

Welcome to Universal Request Analyzer! Follow these simple steps to get started with powerful network request analysis:

  1. Register an Account

    Click the extension icon in your browser toolbar and create your account with an email and password. Your data is stored locally and securely in your browser using SQLite.

  2. View Current Page Activity

    After logging in, the popup shows a real-time summary of requests for the current page including total requests, average response times, status code breakdown, and error detection.

  3. Open DevTools Panel

    Press F12 or right-click ‚Üí Inspect to open DevTools, then click the "URA" tab for advanced charts, comprehensive filters, and detailed request analysis with performance metrics.

  4. Explore the Dashboard

    Click "Dashboard" in the popup to access the full-featured options page with historical data, trend analysis, database inspection, and export capabilities.

Pro Tip: The popup auto-refreshes every 5 seconds to show real-time data. Keep it open while testing to monitor request patterns!

Features Overview

Universal Request Analyzer provides comprehensive network request analysis and monitoring with powerful features designed for developers and QA teams.

Request Capture

Automatically captures all network requests including headers, body, timings, and response data with zero configuration required.

Analytics & Charts

Visualize request data with interactive charts showing performance trends, status distributions, and request type breakdowns.

Performance Metrics

Track DNS, TCP, SSL/TLS timings, P95 response times, and identify performance bottlenecks with detailed timing breakdowns.

Advanced Filtering

Filter requests by status code, method, domain, resource type, performance thresholds, and custom time ranges.

Local SQLite Storage

All data stored securely in SQLite with medallion architecture (Bronze, Silver, Gold) for efficient analytics.

Export Data

Export requests in JSON, CSV, or HAR format for external analysis, reporting, or sharing with team members.

Quality Metrics

Monitor availability, performance, reliability, and security scores with automated quality assessments.

Historical Data

Access historical request data with multiple timeframes from 1 minute to 1 month for trend analysis.

Error Detection

Automatically detect and highlight failed requests (4xx/5xx) with detailed error information for quick debugging.

DevTools Integration

Seamless integration with Chrome DevTools providing a dedicated panel with comprehensive analysis tools.

Theme Support

Multiple theme options (Light, Dark, High Contrast, Blue) with automatic theme detection and switching.

Database Inspection

Direct access to SQLite tables with query history, table previews, and SQL injection protection.

Note: Some features require specific permissions. Make sure to grant necessary permissions when prompted for full functionality.

Frequently Asked Questions

Q: Is my data secure?
Absolutely! All your data is stored locally in your browser using SQLite. Nothing is sent to external servers unless you explicitly enable backend sync in the future. Your sensitive request data, headers, and credentials stay on your machine.
Q: How do I clear my data?
Go to the Dashboard (Options page) by clicking the extension icon ‚Üí Dashboard. Navigate to the Advanced tab where you'll find the "Clear Data" button in the database management section. You can also export your data before clearing for backup purposes.
Q: What browsers are supported?
Universal Request Analyzer currently supports Chrome, Edge, Brave, Opera, and other Chromium-based browsers. Firefox support is planned for future releases as we adapt the extension to use Firefox's WebExtensions API.
Q: Can I use this for team collaboration?
Team collaboration features with backend sync and cloud storage are currently in development. The extension works locally now, but we're building REST API integration for data sharing, team dashboards, and collaborative analysis features.
Q: How much storage does it use?
Storage usage depends on your browsing activity and data retention settings. The extension uses efficient SQLite storage with automatic cleanup of old data. Typical usage is 10-50MB, but you can configure retention periods in settings to manage storage. You can also manually clear data anytime.
Q: Can I export my data?
Yes! Export your data in multiple formats:
  • JSON - For programmatic analysis and integration
  • CSV - For spreadsheet analysis in Excel or Google Sheets
  • HAR - HTTP Archive format for import into other tools
Export from the Dashboard with optional filters for specific time ranges, domains, or request types.
Q: Does it affect browser performance?
The extension is optimized for minimal performance impact. Request capture runs efficiently in the background using Chrome's native APIs. You can also configure sampling rates in advanced settings to reduce overhead if needed. Most users experience no noticeable performance impact.
Q: How do I access the DevTools panel?
Easy! Press F12 (or Cmd+Option+I on Mac) to open Chrome DevTools, then click on the "URA" or "Request Analyzer" tab at the top. This panel shows detailed analysis with charts, filters, and performance metrics for the current page.
Q: Can I track specific APIs or endpoints?
Yes! Use the Advanced Filtering feature in the DevTools panel to filter by:
  • Specific domains or URLs
  • Request types (XHR, Fetch, etc.)
  • Status codes (2xx, 3xx, 4xx, 5xx)
  • Performance thresholds (slow requests)
  • Time ranges
The Endpoints tab also groups similar API calls for easy analysis.
Q: What's the medallion architecture?
The extension uses a medallion (Bronze/Silver/Gold) data architecture:
  • Bronze Layer - Raw request data as captured
  • Silver Layer - Cleaned and enriched data with quality metrics
  • Gold Layer - Aggregated analytics and business metrics
This architecture enables efficient querying and powerful analytics while maintaining data integrity.
