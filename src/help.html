Getting Started

Welcome to Universal Request Analyzer! Follow these simple steps to get started:

  1. Register an Account

    Click the extension icon and create your account with an email and password. Your data is stored locally and securely.

  2. View Current Page Activity

    After logging in, the popup shows a summary of requests for the current page including total requests, response times, and errors.

  3. Open Analytics

    Click "Open Analytics" to access the DevTools panel with advanced charts, filters, and detailed request analysis.

  4. Explore the Dashboard

    Click "Dashboard" to access the full-featured options page with historical data, trends, and export capabilities.

Quick Tips

  • The popup auto-refreshes every 5 seconds to show real-time data
  • All data is stored locally in your browser using SQLite
  • Export your data anytime from the dashboard
  • Use DevTools panel for in-depth analysis of the current page

Features Overview

Universal Request Analyzer provides comprehensive network request analysis and monitoring.

Request Capture

Automatically captures all network requests including headers, timings, and response data.

Analytics & Charts

Visualize request data with interactive charts showing performance trends and patterns.

Performance Metrics

Track DNS, TCP, SSL/TLS timings and identify performance bottlenecks.

Advanced Filtering

Filter requests by status, method, domain, resource type, and more.

Local Storage

All data stored securely in SQLite with medallion architecture for analytics.

Export Data

Export requests in JSON, CSV, or HAR format for external analysis.

Quality Metrics

Monitor availability, performance, reliability, and security scores.

Historical Data

Access historical request data with multiple timeframes (1min to 1 month).

Frequently Asked Questions

Q: Is my data secure?
Yes! All your data is stored locally in your browser using SQLite. Nothing is sent to external servers unless you explicitly enable backend sync in the future.
Q: How do I clear my data?
Go to the Dashboard (Options page) and use the "Clear Data" button in the settings section. You can also export your data before clearing.
Q: What browsers are supported?
Universal Request Analyzer supports Chrome, Edge, Brave, and other Chromium-based browsers. Firefox support is planned for future releases.
Q: Can I use this for team collaboration?
Team collaboration features with backend sync are in development. Currently, the extension works locally, but we're building REST API integration for data sharing between team members.
Q: How much storage does it use?
Storage usage depends on your browsing activity. The extension uses efficient SQLite storage with automatic cleanup of old data. You can configure retention periods in settings.
Q: Can I export my data?
Yes! Export your data in JSON, CSV, or HAR (HTTP Archive) format from the Dashboard. You can export all data or filter specific time ranges.
Q: Does it affect browser performance?
The extension is optimized for minimal performance impact. Request capture runs in the background efficiently, and you can configure sampling rates to reduce overhead if needed.
Q: How do I access the DevTools panel?
Open Chrome DevTools (F12 or right-click → Inspect), then click on the "URA" or "Request Analyzer" tab. This panel shows detailed analysis for the current page.

Troubleshooting

Extension not capturing requests?

  • Make sure you're logged in
  • Refresh the page after installing the extension
  • Check that the extension has necessary permissions

Charts not displaying?

  • Clear browser cache and reload
  • Make sure you have some request data captured
  • Try a different chart timeframe
